Utilisation des filtres

Filtrer une donnée de type "texte"


Ce type de filtre se compose d'un champ dans lequel vous devez renseigner par écrit la donnée que vous souhaitez retrouver. Par exemple, pour retrouver l'utilisateur Admin, il vous suffit de renseigner la valeur "admin" dans ce champ.

Certains filtres textuels s'accompagnent d'un bouton permettant d'activer ou non le mode "phonétique". Ce mode permet de trouver une donnée en se basant uniquement sur la phonétique de la valeur renseignée. C'est-à-dire que si vous tappez la valeur "adeumine" vous obtiendrez la donnée "admin".


Informations complémentaires


Le caractère spécial * représente n'importe quel caractère. C'est-à-dire que si vous renseignez la valeur "ad*" dans ce type de filtre, vous obtiendrez l'ennsemble des données comportant un "ad". Le filtre vous proposera donc des données ayant pour valeur "admin", "administrateur", et cetera.
