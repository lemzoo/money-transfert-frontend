Utilisation des filtres

Filtrer une donnée de type "date"


Ce type de filtre se compose de deux champs dans lesquels vous devez renseigner une date. Pour inscrire une date, deux méthodes vous sont proposées:

  • La première méthode consiste à écrire la date manuellement. Cette dernière doit avoir un format bien précis. Par exemple, le 31 Décembre 2015 s'écrit sous la forme 31/12/2015.
  • La seconde méthode consiste à cliquer sur l'icône qui se trouve à gauche du champ afin d'ouvrir un calendrier virtuel. Il vous suffit donc de sélectionner la date souhaitée dans ce calendier.

Informations complémentaires


Si vous renseignez que la première date, cela signifie que vous cherchez toutes les données à partir de cette date.

Si vous renseignez que la deuxième date, cela signifie que vous cherchez toutes les données jusqu'à cette date.
